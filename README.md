# JobSphere | Next-Gen Job Portal SPA

A premium, full-featured Job Portal frontend built with React, Vite, and Framer Motion.

## 🚀 Key Features
- **Dynamic Job Search:** Powerful filtering by title, company, and location.
- **Role-Based Access:** Personalized dashboards for both Candidates and Administrators.
- **Glassmorphism UI:** State-of-the-art design using modern CSS and backdrop filters.
- **Smooth Animations:** Integrated Framer Motion for premium page and component transitions.
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop.

## 🛠️ Technology Stack
- **Library:** ⚛️ React 18
- **Build Tool:** ⚡ Vite
- **Routing:** 🛣️ React Router 6
- **State Management:** 🧠 Context API
- **Icons:** 🎨 Lucide React
- **Animations:** ✨ Framer Motion

## 📦 Setup & Installation

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run Development Server:**
   ```bash
   npm run dev
   ```

3. **Build for Production:**
   ```bash
   npm run build
   ```
